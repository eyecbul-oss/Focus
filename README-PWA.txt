# SezR Focus PWA

Bu paket Focus sayfasını telefona uygulama gibi eklenebilir hale getirir.

GitHub'a yüklenecek dosyalar:
- focus.html
- focus-style.css
- focus-app.js
- focus-config.js
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

Kontrol:
1. Dosyaları GitHub'a yükle.
2. Siteyi aç.
3. Chrome > üç nokta > Ana ekrana ekle / Uygulamayı yükle.
4. Eğer eski sürüm görünürse tarayıcı önbelleğini temizle veya service worker güncellenene kadar sayfayı birkaç kez yenile.
