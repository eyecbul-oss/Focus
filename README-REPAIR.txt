Bu paket Copilot sonrası oluşan dosya karışıklıklarını temizlemek için hazırlandı.

Düzeltilen ana sorunlar:
- TaskCard.tsx içindeki store kodu kaldırıldı.
- TimerRing.tsx içindeki settingsStore kodu kaldırıldı.
- audioService.ts içindeki navigation kodu kaldırıldı.
- colors.ts içindeki cloud sync kodu kaldırıldı.
- firebase.ts içindeki RootNavigator kodu kaldırıldı.
- NeonButton / GlassCard / GradientBackground tekrar doğru UI bileşeni haline getirildi.
- package.json, app.json ve android-build.yml temizlendi.

Kullanım:
1. ZIP içeriğini repo köküne yükle.
2. Aynı isimli dosyaların üstüne yaz.
3. Eski Capacitor workflow dosyası kalmasın.
4. Actions > Build SezR Focus Expo Android çalıştır.
