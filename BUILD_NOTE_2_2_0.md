# SezR Focus 2.2.0

Bu sürümde Focus uygulamasına ödev ve konu takip paneli eklendi.

Kontrol edilecekler:

- Uygulama çevrimdışı açılır.
- Sınav sayacı çalışır.
- Pomodoro seansı ders istatistiğine işlenir.
- Ödev ekleme çalışır.
- Ödev tamamlama çalışır.
- Ödev silme çalışır.
- JSON yedekleme çalışır.
- CSV dışa aktarma çalışır.

APK build çıktısı GitHub Actions Android CI üzerinden üretilir.
